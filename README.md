# TicTacToe

Tic Tac Toe is an app to play tic tact toe with a friend with 2 different variations.

---

## Getting Started

Clone this repo to your local machine using https://github.com/han-codes/TicTacToe.git

---

### Prerequisites
```
Android Studio installed
```
